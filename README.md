# Opus 4.6 test

## Overview

It should be progressive modern landing page that presents [peacedata](https://peacedata.com) services and products. It is software studio produces cutting-edge software solutions for businesses and organizations.

## Core Goal

You should build 5 landing pages options, that should use different colors, fonts, animations, layout. Be as much creative and inspired as possible. There also should be a buttons to switch between the pages.

## Structure

- NO header, navigation only via context links and footer links
- NO common rectangles, prefer using colors, shadows, diagonal lines, triangles.
- Layout should be responsive and adaptive to different screen sizes, including mobile devices.
- Typography should be modern and clean, should follow the whole design system.
- Do not use pictures, search for CSS animations and patterns to create the visual identity.
- HERO SECTION:
-- Your Last Agency Was a Joke. They played chess. We're here to start a war for making your brand a religion.
-- "Decent results / [peacedata] quality"
- Features / Benefits Section:
-- CHEAP
-- FAST
-- AWESOME
- PORTFOLIO SECTION: (replace with small cards with real sites in cards (make them as regular interactive sites in cards, like new browser tab opened), expandable on hover, user can click and browse the site like it is a real site)
-- Asus router web interface
-- Bank application (mobile + desktop site)
-- Job search service with AI
- Services Section:
-- DEVELOPMENT
-- DESIGN
-- CONSULTING
- (skip testimonials section)
- FOOTER:navigation bar, social links.
